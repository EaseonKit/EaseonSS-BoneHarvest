## Overview
**BoneHarvest** is a lightweight server-side mod that lets you instantly harvest plants using bone items.  
Perfect for quick and convenient resource gathering â€” simply right-click with bone meal, bone, or bone block on supported plants.


## Supported Plants
> **Sugar Cane**  
> **Vine**  
> **Glow Lichen**  
> **Bush**  
> **Firefly Bush**


## Harvest Rates
> **Bone Meal:** 1x plant  
> **Bone**: 3x plant  
> **Bone Block**: 9x plant


## Commands
All commands require **OP level 2** or higher.

**View current status:**
```
/easeon boneharvest
```
**Enable the mod:**
```
/easeon boneharvest on
```
**Disable the mod:**
```
/easeon boneharvest off
```
